Fleet Backup

Stand up a borg backup server from nothing, trust each machine that should back up to it, then add a repository just by naming it. Give a machine a backup job and run it on demand or on the nightly schedule. Vaier emails admins when a run fails, and when the backup server itself goes quiet. Restore isn't available here yet.
